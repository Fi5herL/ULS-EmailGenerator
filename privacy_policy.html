Privacy Policy

Product: Flex Email Generator (Excel)

Last updated: 2026-05-12

1. Single Purpose of the Extension

Flex Email Generator (Excel) is designed for one purpose only: generating standardized Flex Project emails on portal.ul.com using user-provided local .xlsx templates (typically synced via OneDrive).

2. Permissions Used and Why

  • storage: Used to store local user settings and preferences only. This data is never sent to any server.
  • clipboardWrite: Used to copy generated email content to the clipboard for user convenience.
  • activeTab: Used to interact with the active portal.ul.com tab and inject the email generation UI.
  • Host permission: https://portal.ul.com/*: Content scripts are injected on the project details pages to add the email generation button and read project metadata. No data is transmitted externally.

3. Data Collection and Transmission

This extension does not collect, sell, or transmit personal data. All data remains local in the user’s browser and local files.

4. No Remote Code

All JavaScript used by this extension is bundled within the extension package. No remote code is loaded.

5. Third-Party Data Sharing

This extension does not share user data with third parties.

6. Contact

For questions or support, please visit: https://github.com/Fi5herL/ULS-EmailGenerator

1. 擴充功能的單一用途

Flex Email Generator (Excel) 的唯一用途為：在 portal.ul.com 上， 使用使用者提供的本機 .xlsx 範本（通常透過 OneDrive 同步）產生標準化 Flex Project 電子郵件。

2. 權限使用說明

  • storage：僅用於儲存使用者本機設定與偏好，不會傳送到任何伺服器。

  • activeTab：用於與目前啟用中的 portal.ul.com 分頁互動，並注入郵件生成功能介面。
  • 網站存取權限：https://portal.ul.com/*： 僅在專案明細頁注入內容腳本，以新增郵件生成功能按鈕並讀取專案中繼資料；不會對外傳輸任何資料。

3. 資料蒐集與傳輸

本擴充功能不會蒐集、販售或傳輸任何個人資料。所有資料皆保留在使用者瀏覽器與本機檔案中。

4. 無遠端程式碼

本擴充功能所有 JavaScript 均已封裝於套件內，不會載入任何遠端程式碼。

5. 第三方資料分享

本擴充功能不會與第三方分享使用者資料。
